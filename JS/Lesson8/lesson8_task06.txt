Напишите Функция fahrenheitToCelcius для  перевода градусов 
  Фаренгейта в градусы Цельсия. Функция принимает значение температуры в градусах фаренгейта 
и возвращает температуру в градусах Цельсия

 function fahrenheitToCelcius(f){
   return (f-32)/1.8
 }
